    4. *CSS selectors are used to "select" HTML elements and apply styles to them. Understanding different types of selectors allows you to target elements efficiently and write cleaner, more effective stylesheets.

    Selectors are of 11 types

    • Element Selector

      • * it target all the element used in code

      p{ color :red; }


    • Class Selector

      • * it target all the class mention used in element and have to declair with "."

      div class ="box"

      .box { border: 1px solid black; }


    • ID Selector

      • * Apply on the given id and symol of using id is "#"

      div id ="box"

      #box { border: 1px solid black; }


    • Universal Selector

      • * Apply on the given id and symol of using id is "#"

      div id ="box"

      #box { border: 1px solid black; }
