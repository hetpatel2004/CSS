Selectors are of 11 types

Selectors are use to select html elements on which we have to apply style

  1. Element Selector

    2. Meaning: Selects all elements of a specific HTML tag

    we just only have to write element name and {start applying propetties}

  2. Class Selector

    3. Meaning: Selects elements with a specific class attribute
    p{ background-colour:red; }

    have to put "." before writing any attribute

    and it referes to element name nothing else

  3. ID Selector

    4. Selects an element with a specific id and writen like this
    #abc{ background-colour:red; }
    ID must be unique in a page
    and it referes to "#" sine

  4. Universal Selector

    5. Universal selector use to all css in all over the page and writen like this
    *{ margin = 0; padding = 0; }

    and it referes to "*" sine

  5. Grouping Selector

    6. Grouping selector use to allply tings on two or more then that
    h1,h2,h3,{ background-colour:red; }

    and it referes to "," sine

  6. Decident Selector

    7. Targets elements that are nested inside a specific parent.
    div p { color: green; }

    and it referes to " " (space) sine

    mostly used while creating PSD

  7. Child Selector

    8. Targets elements that are nested inside a specific parent.
    ul > li { list-style-type: square; }

    and it referes to ">" sign

  8. Adjacent Sibling Selector

    9. Targets the first element that comes immediately after another.
    h1 + p { color: red; }

    and it referes to "+" sign

  9. General Sibling Selector

    10. Selects all siblings after a specified element.
    h1 ~ p { font-style: italic; }

    and it referes to "~" sign

  10. Pseudo-Classes Selector

    11. Define a special state of an element (hover, active, etc).
    h2:hover { color: orange; }

    and it referes to ":" sign

  11. Pseudo-Element Selector

    12. Style specific parts of an element.
    p::first-line { font-weight: bold; }

    and it referes to "::" sign

  12. Atribute Selector

    13. Apply on any atribute
    input[required] { border: 2px solid red; }

    and it referes to "[]" sign
