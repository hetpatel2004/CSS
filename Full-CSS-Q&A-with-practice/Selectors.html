Selectors are of 11 types

Selectors are use to select html elements on which we have to apply style

  1. Element Selector

    2. Meaning: Selects all elements of a specific HTML tag

    we just only have to write element name and {start applying propetties}

  2. Class Selector

    3. Meaning: Selects elements with a specific class attribute

    have to put "." before writing any attribute

  3. ID Selector

    4. Meaning: Selects an element with a specific id
    Note: ID must be unique in a page
